# MotifNusa - Pengenalan Batik Indonesia

Template sederhana dibuat dengan Vite + React + Tailwind CSS.

Cara pakai

1. `npm install vite` Kalo ngoni blm ada vite install dulu
2. `npm run dev` Supaya program jalan di website local

Logo & gambar ada di `public/images/`.

Note :

1. Jangan lupa tambahkan gambar tiap daerah beserta keterangganya
2. Tambahkan keterangan di About.jsx supaya lebe estetik
